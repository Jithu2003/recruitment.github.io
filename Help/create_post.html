Create Post [ Help Index ]

The Create Post Utility allows the Administrator to create a Post ie. the job position in the company for which candidates should be recruited. The Post is created as per the pattern specified in the Post specifications according to the requirments of the company. Each Post is unique and the system will prompt if the post name specified in the create post is already present. The candidates can apply to the post if and only if there is atleast one vacant position for that post and there is atleast one category of test present in the Post.

The management of the company can decide the specifications for the job position depending on their requirements. Then the system automatically checks the eligibility of the applying candidate and sorts out the desired candidates to be selected in the view selected candidates.

This Utility is only available for those Administrators who have the authority to Create Posts and Manage Candidate Selection. See the Accessibility Options for more details.

Warning : It is better that the Administrator should first Shut the User Server before he creates a Post.
