The Create Category Utility allows the Administrator to create a Category ie. the subject for the test to be taken for a particular Post.The Category has the least fields of all with just the category name and the description. Every Category is identified by its name which is unique. The name of the Category is entered according to the prescribed Name Field standards and as specified in the Category specifications. If the user enters the category name which is already present in the database then the system will prompt to enter an alternative category name.

This Utility is only available for those Administrators who have the authority to Create Categories and Manage Exams. See the Accessibility Options for more details.

Warning : It is better that the Administrator should first Shut the User Server before he creates a Category.
