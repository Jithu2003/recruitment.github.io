Create Test [ Help Index ]

The Create test Utility allows the Administrator to create a Test pattern. This Test pattern would later be used to Publish the test for a particular Category. Each Test created is unique by its name event the settings are same for more than one test. So if the user enters the name of the test which is already present in the database then the system will prompt to enter an alternate test name. One can skillfully create such customized Test patterns and reuse them for more than one tests.

This Utility is only available for those Administrators who have the authority to Create Tests and Manage Exam. See the Accessibility Options for more details.

Warning : It is better that the Administrator should first Shut the User Server before he creates a Test.
