Category [ Help Index ]

The Category is the Subject for which the candidate or the User has to give the test. The Questions are created for the Category and it the Category which decides the subject or the content of the Questions. Hence Category plays an key role in the Exam Management. With the help of the Category one can Publish Test by linking the Category to the corresponding Test pattern.

The attributes of the Category include the following :

 Category  Name This is the name of the test and must be unique. It is based on the conventions of Name Field and should not exceed more than 20 characters. 
 Description  This is the description of the test of general information. The total characters should not exceed more than 500.
